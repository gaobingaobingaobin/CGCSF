CGCSF SOFTWARE RELEASE

========================================

Objective:

Generates the artifact-visibility (local visual masking thresholds) of an artifact when over-layed on an input natural image. Uses contrast gain control and structural facilitation.

========================================

Usage: 

Please run the demo.m in Matlab. Code is well commented.
